Every Picture

10/27/2021

When the user slide to the set, that set will light up. In addittion, the modal will distort toward where your mouse moves, which really grabs users' attention and encourage users to focus on the set. When clicking into the set, the modal has a live picture where it can rotate up to 90 degree like VR. Other website only offers still side or front view. When the user scrolls down, there is a fluent flow in images that direct users' eyes from top to down.
